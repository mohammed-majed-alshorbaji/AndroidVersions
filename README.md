# تطبيق إصدارات أندرويد | AndroidVersions App

هاد تطبيق أندرويد بيعرض قائمة كاملة بـ 12 اصدار من اصدارات نظام الأندرويد مع الشعارات الرسمية والجديدة تاعتهم. التطبيق معمول بشكل سلس في الحركة، وفيه ميزة بتخليك ترتب العناصر حسب الحروف الأبجدية بضغطة زر وحدة.


# الميزات اللي عملتها في المشروع:

قائمة مرنة وسريعة (RecyclerView):استخدمت الـ RecyclerView عشان تعرض كل الـ 12 إصدار بشكل سريع وبدون أي تعليق لما تعمل سكرول (تمرير) لفوق ولتحت.
تصميم بطاقات احترافي (CardView): صممت واجهة مخصصة لكل عنصر في القائمة جوا بطاقة شكلها مرتب، وبتعرض اسم الإصدار، رقمه، وشعاره.
شعارات مخصصة وشفافة:ضفت كل الصور والشعارات بصيغة شفافة وبخلفيات متناسقة جوا مجلد الـ drawable عشان تطلع واجهة المستخدم (UI) مرتبة ونظيفة.
ترتيب أبجدي بضغطة زر: ضفت زر علوي اسمه (Sort Alphabetically)، أول ما تضغط عليه برتب كل القائمة حسب الحروف الأبجدية فوراً وبدون ما تحتاج تطفي التطبيق وتشغله تاني.
